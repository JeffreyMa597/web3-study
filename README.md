# 安装配置

### 下载NodeJs
地址:  [NodeJs 官网](https://nodejs.org/zh-cn/ "NodeJs 官网")


### 下载目录
提供一下两种方式下载
1. 直接通过压缩包     [[压缩包]](https://github.com/OkamotoSatoshi/web3-study/archive/refs/heads/master.zip "[压缩包]")
2. git clone 方式
`
 git clone git@github.com:OkamotoSatoshi/web3-study.git
`

### 在根目录下安装
安装命令  npm install 

### 运行开发环境
npm run dev

### 浏览器中打开
http://localhost:3000
